# Capitol geometry references

Primary dome profile reference:

- Library of Congress item `93510472`, Thomas U. Walter / W. B. Franklin,
  *U.S. Capitol Dome, Cupola Roof: Plan, Elevations, Sections, and Details*,
  circa 1860–1865.
- Rights advisory: no known restrictions on publication.
- Local filename: `loc_93510472_capitol_dome_cupola.tif`.
- Source: https://www.loc.gov/item/93510472/
- The drawing note specifies 36 hollow cupola-roof ribs. The exterior generator
  uses that documented count for both visible curved ribs and shell panel bays;
  this is a silhouette constraint, not a claim that the procedural rib profile
  is a final fabrication-accurate mesh.

Overall building dimensions are checked against the Architect of the Capitol:
751 ft 4 in north–south length, 350 ft greatest width including approaches,
and 288 ft from the East Front baseline to the top of the Statue of Freedom.
