# jan6game

Unreal-ready public/schematic U.S. Capitol map assets and supporting tools.

Main package:

- `capitol_unreal_map/` - generated OBJ meshes, metadata, textures, viewer, and Unreal import scripts.

Useful commands:

```bash
python3 capitol_unreal_map/scripts/build_capitol_unreal_map.py
python3 capitol_unreal_map/scripts/validate_capitol_package.py
```

Detailed docs live in `capitol_unreal_map/README.md`.
